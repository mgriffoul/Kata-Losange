# Kata-Losange

## Kata Description
Ecrire en programme en javascript qui : 

Etant donnée une lettre, affiche un losange commencant par 'A', et ayant la lettre donnée aux sommets centraux.

### Examples
```
Input: A
Output: A

Input: B
Output:   A
         B B
          A


Input: D
Output:    A
          B B
         C   C
	D     D
         C   C
          B B
           A
```
